5. Test Refresh & Favoris

Test manuel :

  1. Cliquez sur n'importe quel lien ci-dessus
  2. Une fois la page chargée, appuyez sur F5 (refresh)
  3. La page doit se recharger correctement (pas d'erreur 404)
  4. Ajoutez la page en favoris (Ctrl+D)
  5. Rouvrez le favori - doit fonctionner
