# Hero Assets

Ce dossier contient les images utilisées dans la section hero de la page d'accueil.

## Fichier requis

- `image_hero.gif` - Image GIF animée utilisée comme arrière-plan de la section hero

## Important

L'image doit être placée dans ce dossier pour être accessible en production.
Le chemin utilisé dans le code est : `/assets/hero/image_hero.gif`

## Taille recommandée

- Résolution : 1920x1080 ou supérieure
- Format : GIF animé optimisé
- Poids : < 5MB pour de meilleures performances
